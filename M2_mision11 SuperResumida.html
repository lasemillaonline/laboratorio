Trayecto inicial · Misión 11

Encender un LED desde la web

En esta misión vas a lograr que un botón en la página web controle una acción real en la placa. Cuando hagas clic, el LED rojo D8 se va a encender o apagar.

1. Propósito de la misión

El objetivo es unir la página web con una salida física de la placa para ver un resultado real.

Meta simple: tocar un botón en la web y ver que el LED rojo responde.

2. Qué debe hacer el alumno

  1. Abrí tu proyecto en Steamakers Blocks.
  2. Revisá que la red WiFi y la página web de la misión anterior funcionen bien.
  3. Agregá en la página un botón para encender el LED rojo D8.
  4. Agregá otro botón para apagar el LED rojo D8.
  5. Buscá la forma de crear una acción o ruta distinta para cada botón.
  6. Relacioná cada acción web con el pin D8 de la placa.
  7. Cargá el programa en la placa.
  8. Conectate a la red WiFi de la Wemos D1.
  9. Abrí la página desde el navegador.
  10. Probá los dos botones hasta comprobar que el LED se enciende y se apaga.

3. Consulta sugerida para hacerle a la IA

Prompt para copiar y pegar:

Estoy trabajando con una Wemos D1 en Steamakers Blocks. Ya tengo una página web local funcionando y ahora quiero agregar dos botones: uno para encender el LED rojo D8 y otro para apagarlo. Explicame en pasos muy simples qué bloques debo usar, cómo separar las acciones del servidor web y cómo hacer para que el clic en la página provoque una acción real en la placa. Escribilo para un principiante.

4. Plantilla de trabajo para completar

Completá esta ficha y después apretá Descargar en PDF.

5. Criterio de entrega

  • La página debe tener dos botones claros: encender y apagar.
  • El LED rojo D8 debe responder a esas acciones.
  • La plantilla debe estar completa.
  • Debés poder explicar con palabras simples qué pasó entre el clic y la reacción del LED.

6. Ejemplo de respuesta simple

Qué entendí de esta misión: Entendí que la página web no solo muestra información, también puede enviar órdenes a la placa.

Resultado obtenido: Desde mi celular toqué el botón de encender y el LED rojo se prendió. Después toqué apagar y se apagó.

7. Guion del video (60 a 90 segundos)

Inicio: Decí tu nombre y nombrá la misión.

Demostración: Mostrá el clic en la página y la respuesta del LED rojo D8.

Explicación final: Contá qué le pediste a la IA, qué bloques agregaste y cómo entendiste el recorrido entre la web y la placa.

8. Cierre motivador

Muy bien. Acabás de lograr algo clave: controlar una acción física desde la web. Este es uno de los momentos más importantes del proyecto, porque ya uniste interfaz digital y respuesta real.
