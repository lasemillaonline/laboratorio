Misión 5: Mostrar en qué modo está tu sistema

En esta actividad vas a usar los 3 LEDs y el pulsador para que la placa muestre estados de forma clara. La idea es simple: cada luz representa un modo distinto, y el botón sirve para pasar de un modo a otro.

1. Propósito de la misión

Vas a aprender a organizar el sistema por modos o estados. Eso significa que cada LED tendrá un significado.

Al terminar, vas a poder apretar el pulsador y ver cómo cambia el modo del sistema con una luz distinta.

Rojo = Espera
Azul = Manual
Verde = Automático

2. Qué debe hacer el alumno

  1. Elegí 3 modos simples para tu sistema.
  2. Asigná un LED a cada modo.
  3. Creá una variable llamada estado.
  4. Programá el pulsador D2 para que cambie al siguiente estado cada vez que se presiona.
  5. Hacé que solo se encienda el LED correspondiente al estado actual.
  6. Probá varias veces hasta que el cambio sea claro y ordenado.
  7. Explicá con tus palabras qué significa cada luz.
Idea simple: Estado 1 = espera, Estado 2 = manual, Estado 3 = automático.

3. Consulta sugerida para hacerle a la IA

Ayudame a programar en bloques una Wemos D1. Quiero hacer una misión simple con 3 estados. Condiciones: - Pulsador en D2 - LED rojo en D8 - LED azul en D7 - LED verde en D6 - Cada vez que aprieto el pulsador, el sistema cambia de estado - Estado 1: se enciende LED rojo - Estado 2: se enciende LED azul - Estado 3: se enciende LED verde - Después del estado 3, debe volver al estado 1 Explicame los bloques paso a paso, con palabras simples, para un principiante.

4. Criterio de entrega

El alumno debe entregar:

  • La plantilla completa desde esta página.
  • Un video corto donde se vea el cambio de estados con el pulsador.
  • Una explicación breve de qué representa cada LED.

La misión está lograda si al presionar el pulsador el sistema cambia de modo y cada modo se muestra con una luz distinta.

5. Guion del video (60 a 90 segundos)

  1. Inicio: Decí tu nombre y el nombre de la misión.
  2. Demostración: Mostrá la placa y apretá el pulsador para cambiar de estado.
  3. Explicación: Contá qué le pediste a la IA, qué bloques usaste y qué significa cada LED.
  4. Cierre: Decí si funcionó bien o qué tuviste que corregir.

6. Plantilla de trabajo para completar

Para entregarla en PDF, hacé clic en Descargar / Guardar en PDF y elegí Guardar como PDF en la ventana de impresión.
