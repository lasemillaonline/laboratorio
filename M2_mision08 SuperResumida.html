Misión 8: Dejar listo y ordenado el sistema local

En esta misión no agregás cosas nuevas. Lo importante es revisar, ordenar y dejar funcionando bien todo lo que ya hiciste. La meta es tener una versión local estable antes de pasar a la web.

Nivel: principiante
Objetivo: ordenar y mejorar
Con ayuda de IA
Lista para PDF

1. Título simple de la misión

Dejar listo y ordenado el sistema local

2. Propósito de la misión

Vas a revisar tu programa para que quede más claro, más limpio y más estable.

  • Ordenar bloques.
  • Mejorar nombres de variables.
  • Sacar repeticiones innecesarias.
  • Probar que todo funcione varias veces.

Al final, tu sistema tiene que quedar listo para seguir con la etapa web.

3. Qué debe hacer el alumno

  1. Abrí el proyecto que hiciste hasta la misión 7.
  2. Revisá si hay bloques repetidos o desordenados.
  3. Cambiá los nombres de variables para que se entiendan mejor.
  4. Dejá clara la parte del modo manual y la del modo automático.
  5. Probá el pulsador, el sensor y los LEDs varias veces.
  6. Corregí errores pequeños hasta que el sistema responda siempre bien.
  7. Mostrá que tu versión 1 quedó estable y lista para seguir.
Consejo: esta misión no se trata de hacer más. Se trata de hacer mejor lo que ya tenés.

4. Consulta sugerida para hacerle a la IA

Estoy trabajando con una Wemos D1 en Steamakers Blocks. Ya tengo un sistema local con pulsador, sensor de luz, LEDs y dos modos: manual y automático. Quiero dejar mi versión 1 más ordenada y más fácil de entender. Ayudame a: 1) detectar bloques repetidos 2) mejorar nombres de variables 3) simplificar la lógica 4) revisar qué debería probar antes de pasar a la web Explicamelo con palabras simples y paso a paso.

5. Criterio de entrega

Tenés que entregar:

  • Esta plantilla completa.
  • Un video corto de 60 a 90 segundos.
  • La demostración de la versión 1 funcionando de forma estable.

Se considera cumplida si se ve claramente que:

  • el sistema está más ordenado que antes,
  • los modos se entienden mejor,
  • el pulsador, el sensor y los LEDs funcionan sin confusión.

Parte 3

Parte 4

8. Ejemplo simple de respuesta

Qué entendí: tenía que ordenar y revisar todo antes de avanzar.

Qué le pedí a la IA: le pedí ayuda para cambiar nombres de variables y simplificar bloques repetidos.

Resultado: dejé más clara la parte manual y la automática, y el sistema quedó funcionando mejor.

9. Cierre motivador

Ahora tenés una base más firme para dar el siguiente paso: conectar tu proyecto con la web sin empezar de cero.
